Virtual DMZ

Virtual DMZ(De Militarized Zone) allows you to specify a DMZ host IP to redirect your request to Virtual DMZ host in order to enhance the security of the local area network.

  » Virtual DMZ (Enable/Disable): Enable or disable the Virtual DMZ function. If the function is enabled, the threat from external network will be directed to the Virtual DMZ instead of the network.
  » IP Address of host DMZ: Enter the IP address of host DMZ.
