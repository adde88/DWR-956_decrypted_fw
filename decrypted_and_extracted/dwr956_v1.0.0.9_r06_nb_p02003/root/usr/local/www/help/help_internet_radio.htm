Radio Configuration

This page allows you to set LTE related configurations.

  » Radio Mode: Select 2G or 3G or 4G for your mobile network or select 4G Preferred to switch between available 2G/3G/4G mobile networks automatically.
  » Radio Frequency: Select a radio frequency according to your location.
  » Antenna: Select EXT/INT to turn on/off the outdoor antenna.
