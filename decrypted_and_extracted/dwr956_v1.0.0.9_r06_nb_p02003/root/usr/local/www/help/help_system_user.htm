Users

This page lists the users that have been configured for accessing shares, the Internet or even managing the device. The default guest or admin user is authorized to access the web management interface of this product. A newly configured user (along with the default guest / admin) can be assigned to a group, upon which common parental control and content sharing policies can be applied. Custom users are not able to access the web management interface.

Click Add to add a user account, or select a user and click Edit to modify its contents:
  » Username: Unique identifier for this user.
  » Password: The password of the user.
  » Confirm Password: Enter the password again.
  » Description: Description of the user.
  » Group Membership: Group to which the user belongs.
  » Idle Timeout: the time that the user will be logged out.

To delete a user, select the user and click the delete icon to delete the selected item.
