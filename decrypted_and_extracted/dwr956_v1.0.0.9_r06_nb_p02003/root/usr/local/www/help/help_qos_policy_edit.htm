Bandwidth Profiles

Bandwidth Profiles: Apply a priority profile for a designated computer when using a certain service. Click Add to add a rule:
  »» Available Profiles: The priority in which profiles will be applied.
  »» Service: The range of the IP address set in the profile.
  »» Traffic Selector Match Type: Priority of the bandwidth profile.
  »» IP Address: The IP address to which this profile applies, for example, the IP address of your gaming computer or VoIP phone.
  »» DSCP Value: The profile will apply to packets with this DSCP value, for example, the DSCP value you set for a smartphone.
