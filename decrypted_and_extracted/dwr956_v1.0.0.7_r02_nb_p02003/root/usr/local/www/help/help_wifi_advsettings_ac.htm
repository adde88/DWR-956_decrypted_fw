Wi-Fi Advanced-> 802.11ac

This section allows you to configure and manage advanced settings of your wireless network in 802.11a/n/ac modes. In most cases, however, the default wireless settings will be sufficient.
  » Mode: Select the 802.11 modulation technique. This device supports 802.11 b/g/n Wi-Fi modes; the available modes are dependent on the operating frequency selected.
The available modes for the 2.4 GHz setting are:
802.11a: Select this mode if all devices in the wireless network can support 802.11a.
802.11a/n: Select this mode to allow devices supporting 802.11a or 802.11n to connect to the product.
802.11ac: Select this mode if all devices in the wireless network can support 802.11ac.

  » Channel Spacing: This is the difference in frequency between successive radios.
  » Channel: This specifies the frequency the radio uses to transmit the wireless frames. Select a channel from the list of channels.
  » Transmit Power: Select the signal power strength of the product's Wi-Fi network. Click Apply to apply the changes, or click Reset to undo your configurations.

Click Apply to save your changes.
Click Reset to revert to the previous settings.
