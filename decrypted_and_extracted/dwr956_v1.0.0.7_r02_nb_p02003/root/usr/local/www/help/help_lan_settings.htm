LAN Settings

This configuration page allows you to configure the local network settings of this product.
Note: The settings on this page will affect all devices that are connected to the LAN of this product, including wireless LAN clients. All connected LAN hosts must be in the same subnet and use the new LAN address to access the user configuration Interface.

  » IP Address: Enter the local network IP address for this product. The default IP address is 192.168.1.1.
  » Subnet Mask: The subnet mask along with the previously configured IP address defines the network. The default value for subnet mask is 255.255.255.0.
  » Host Name: Host name of the product.

Click Apply to apply the changes. Or click Reset to undo your configurations.
