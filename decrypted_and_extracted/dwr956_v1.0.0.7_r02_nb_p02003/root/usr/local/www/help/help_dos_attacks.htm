DoS Attacks

A denial-of-service attack (DoS attack) is an attempt to make a device resource unavailable to its intended users.

  » SYN Flood: Enter the maximum number of SYN packets per second the security appliance accepts before determining that a SYN Flood Intrusion is occurring. This value can range between 1 and 10,000 SYN packets per second. The default is 128 SYN packets per second.
  » Echo Storm: The security appliance monitors the number of pings per second to determine when to declare an echo storm intrusion event. Echo storm intrusion events are not blacklisted. This value can range between 1 and 10,000 ping packets per second. The default is 128 ping packets per second.
  » ICMP Flood: The security appliance monitors the number of ICMP packets per second, not including ping packets, to determine when to declare an ICMP flood intrusion event. ICMP flood events are not blacklisted. This value can range between 1 and 10,000 ICMP packets per second. The default is 128 ICMP packets per second.
