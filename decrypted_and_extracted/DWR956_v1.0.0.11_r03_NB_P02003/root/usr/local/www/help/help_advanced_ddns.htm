DNS & DynDNS

The Domain Name Server (DNS) page allows you to define the system name and domain name of this product for DNS resolution. You can assign a domain name to a particular computer with the usage of Domain Name System (DNS) entries. DNS allows you to assign a readable system name to this product. All connected local network computers will appear in the list of hosts as belonging to the domain configured on this page. Network hosts which are a part of a common domain can view the list of other computers that are part of that domain.

  » Domain Name Server (DNS) Address:
  »» System Name: Enter the system name for this product.
  »» Domain Name: Enter the domain name to define the common domain for computers connected to this product.
  »» Description: Add some descriptions about the system name.

  » Dynamic DNS (DDNS) is an Internet service that allows routers with varying public IP addresses to be located using Internet domain names. To use DDNS, you must setup an account with a DDNS provider such as DynDNS.org and fill in the required account details including Domain Name, Username and Password on this page.
  »» Domain Name: Enter the domain name of the dynamic DNS account. This is the DNS domain name which will be used. An example is dyndns.org.
  »» Username: Enter the username of the dynamic DNS account. This will be provided by the dynamic DNS service provider.
  »» Password: Enter the password for the dynamic DNS account.
  »» Use wildcards: Check this option to use wildcards.
  »» Update every 30 days: Selecting this configures this product to update the host information on DynDNS.com and keep the subscription active after the 30 day trial period.
