Remote Management

This feature can be used to manage the device remotely from the WAN side. By default, Remote Management is disabled. To enable WAN access to the Configuration Utility, check the box.
IMPORTANT: When Remote Management is enabled, the security appliance is accessible to anyone who knows its IP address. Since a malicious WAN user can reconfigure this product and misuse it in many ways, it is HIGHLY RECOMMENDED that you change the admin and guest passwords before continuing.
Permission for Remote Management can be given to the following:
  » Enable Remote Management: Check the box to enable remote management.
  » Access Type: Select an access type. If you choose All IP Addresses, this product will allow access of all IPs automatically. If IP Address Range is selected, you will have to specify the starting and ending IP addresses from the following column. If Only this PC is selected, you will have to specify your PC IP address.
  » IP Address: IP address of the PC given remote management permissions

  » Multiple IP Range Configuration
  »» Start IP Address: Displays the starting IP address for the allowed range.
  »» End IP Address: Displays the ending IP address for the allowed range.
