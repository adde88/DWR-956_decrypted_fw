Wi-Fi Protected Setup (WPS)

  » Wi-Fi Protected Setup (WPS): WPS is a computing standard for easy and secure setup of a wireless connection. In this section you can enable WPS for a client connection using OPEN/WPA/WPA2 security.
1. Check the box to enable WPS.
2. Select the SSID of the wireless network that you want to establish a connection via WPS.
3. Click Apply to apply the changes. Or click Reset to undo your configurations.

  » Push Button Connect (PBC): If other wireless devices support the PBC WPS function, Click the "Click for PBC" and then push the WPS button on the other device.

  » Personal Information Number (PIN): If other wireless devices support the PIN WPS function, specify the same PIN for the product and the wireless client device and then click the Via PIN icon to establish a connection.
