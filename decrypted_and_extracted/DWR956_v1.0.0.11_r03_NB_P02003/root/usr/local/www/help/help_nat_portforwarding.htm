Port Forwarding

Port Forwarding can be used to translate the common service port to a custom port inside your local network such as web or FTP.
  » List of Port Forwarding Services: Displays all the port forwarding rules for this product.
  »» Service: The name of the service for which the port forwarding rule has been created.
  »» IP Address: The IP address of the computer on the local network to which the traffic will be forwarded.
  »» Port Translation: The port number of the destination host for traffic forwarding.

Click Add to add a port forwarding rule, or select an item and click Edit to modify or click Delete to delete the selected item.
