PIN Configuration

PIN Configuration allows settings of the PIN protection of your SIM card.

  » Enable PIN Protection: Choose to enable or disable PIN protection. If you choose YES, you will be asked to enter the PIN code when you use the product.
  » PIN Verify: Enter the PIN code.
  » Change PIN: When PIN Protection is enabled, you can click this link to change your PIN code. Enter your current PIN code and new PIN code, and then click the Apply button after confirming your new PIN code.
  » PUK Verify: When you enter the PIN code incorrectly three consecutive times, your SIM card will be locked and you will need a PUK code to unlock it.
