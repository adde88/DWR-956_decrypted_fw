By default, this product will function as a DHCP (Dynamic Host Configuration Protocol) server. As such it will provide TCP/IP configuration to client computers connected to the local network.

  » IPv4 Address Distribution:
  »» Specify the number of devices/computers in your network: Enter the number of local network computers (including those connecting over WiFi) and the DHCP server will calculate the number of leases required for the network.
  »» Start IP distribution from: Define the starting range of the IP address. The DHCP server will iterate the DHCP IP addresses to be given out by taking into account the starting IP address range and the configured number of devices in your network.
